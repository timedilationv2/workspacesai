# SophieAI React App

## Setup

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

- Push to GitHub:
  ```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <YOUR_GITHUB_URL>
git push -u origin main
```

- Deploy on Vercel:
  1. Install Vercel CLI: `npm i -g vercel`
  2. Run `vercel` and follow prompts
